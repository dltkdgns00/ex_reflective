# EX_Reflective_Programming

반영 프로그래밍, wsgi, flask, websocket을 이용한 웹 서버 구축

## 반영 프로그래밍이란?

반영 프로그래밍은 객체지향 프로그래밍의 한 방법으로, 객체의 속성을 변경할 때마다 객체의 상태를 반영하는 방식을 말합니다.
[참고 - Wikipedia](https://en.wikipedia.org/wiki/Reflective_programming)
